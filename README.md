# PokeAPI
Bu uygulama, Python ve Tkinter kullanılarak geliştirilmiş basit bir masaüstü uygulamasıdır. PokeAPI üzerinden Pokémon bilgilerini çekerek her buton tıklamasında farklı bir Pokémon’un adı, ID’si ve türlerini kullanıcıya gösterir.  Uygulamanın amacı, API kullanımı, HTTP istekleri ve temel GUI geliştirme konularını pratik etmektir.
